The challenge is to paint each cell black or white, subject to the following rules:
- Each numbered cell is an island cell, the number in it is the number of cells in that island .
- Each island must contain exactly one numbered cell
- There must be only one sea, which is not allowed to contain "pools", which means 2x2 areas of black cells
Press NEW GAME to start a game, press reset to remove every gray squares, press check when you think you have completed the game.
